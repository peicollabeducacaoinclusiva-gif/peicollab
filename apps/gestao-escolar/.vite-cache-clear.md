# Como Limpar Cache do Vite

Se você encontrar erros de resolução de módulos como "Failed to resolve import 'zod'", siga estes passos:

1. Pare o servidor de desenvolvimento (Ctrl+C)

2. Limpe o cache do Vite:
   ```powershell
   Remove-Item -Recurse -Force node_modules\.vite
   ```

3. Reinicie o servidor:
   ```powershell
   npm run dev
   ```

O cache do Vite será recriado e os módulos serão resolvidos corretamente.
